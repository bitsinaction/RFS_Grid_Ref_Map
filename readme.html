Rural Fire Service Grid Reference Map Plotter - https://bitsinaction.github.io/RFS_Grid_Ref_Map/

Plot a 6 digit grid reference on a map.

A RFS 6 digit grid reference, is a shorthand, fast and simplified mwy of communicating locations within a operations area.

This simplification involves removing redundant (to local operations) data, such as uneccessary accuracy, and the broader location.

As such the operations area that the grid reference applies to is not know, just from the 6 digit ref. So this map just plots all possibilities (for each blue grid), and you must work out which grid they are likely talking about from the location of the reporter.

To get a 6 digit reference, RFS start with a full UTM grid reference off a map and:
For the Easting, take the 2nd,3rd & 4th digits, and for the Northing take the 3rd, 4th & 5th digit.
